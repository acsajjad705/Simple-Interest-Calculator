# Simple Interest Calculator

## Overview
This repository documents a Simple Interest Calculator. It explains the formula, shows example inputs and outputs, and points to an example implementation.

## Formula
Simple Interest (SI) is calculated as:


\[
\text{SI} = \frac{P \cdot R \cdot T}{100}
\]


where **P** = principal, **R** = annual rate in percent, **T** = time in years.

## Example
Input: P = 1000, R = 5, T = 2  
Output: Simple Interest = 100; Total Amount = 1100

## Files
- `README.md` — documentation.
- `calculator.py` — example Python implementation.

## How to run (Python)
1. Install Python 3.
2. Run: `python3 calculator.py`
3. Enter principal, rate, and time when prompted.
