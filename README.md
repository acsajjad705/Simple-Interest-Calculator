# Simple Interest Calculator

## Overview
This repository contains a simple interest calculator that computes interest using the formula:
SI = (P * R * T) / 100

## Usage
1. Provide **P** (principal), **R** (annual rate in %), and **T** (time in years).
2. The program outputs the simple interest and the total amount (principal + interest).

## Example
Input: P=1000, R=5, T=2  
Output: Simple Interest = 100; Total Amount = 1100
